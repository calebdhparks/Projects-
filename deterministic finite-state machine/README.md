# Create class to model a DFA
## Simulating a DFA
./DFA.py -s <DFA_File> <Strings_File>
## Build DFA from Text 
./DFA.py -t <String_File>
## Make Complement of DFA
./DFA -c <DFA_File>
## Find intersection 
./DFA -c <DFA_File1> <DFA_File2>
## Find inverse of DFA 
./DFA.py -i <DFA_file> <Homoorphism_file>
